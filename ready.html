1) Запись ascii НЕ используется вместе с квадратной записью! - ascii мы используем только тогда, когда есть спека от клиента, либо от сервиса, и когда стандартный хак не отработал и на аутлуках фонт с засечками - если у тебя гугл фонт, кастомный или helvetica neue (подключаем линку и импорт) - для хельветики нюе с записью ascii как в таблетках, используем квадратную запись без линки и импорта - записываем название фонта на тд с пробелами и по дефолту без кавычек. Кавычки используем, если где-то баг есть - Если у тебя 500 фонт (добавляем квадратную скобку только для 500. Сам фонт не нужен) 2) Длинную запись можно и со стандартным хаком писать, без анси 3) Любая кастомная хельветика (LT, Pro и др.) заменяем на обычную Helvetica 4) Если клиент просит подключить фонт без тайпкита, то мы оставляем только квадратные скобки и обычную запись - то-же самое для ascii (для хельетики нюе следи что-бы семейство начиналось не с Ариал а с Хельветики) .btn-01:hover{background:#44D484!important;} .btn-01 a{color:#fff; text-decoration:none; display:block; padding:8px 15px;} .btn-01:hover a{text-decoration:none!important;} .l-fix a{color:#006db1; color:#006db1 !important; text-decoration:none !important;} - установи git - введи данные профиля git config --global user.name "name" потом git config --global user.email "email" - создай папку на компе. где угодно под любым именем. в ней будут хранится наши репозитории - зайди в эту папку и нажми shift+ПКМ -> октрыть окно powershell (можно и через обычную консоль перейти в эту папку) - дальше в консоле набери - git clone https://git.twogether.io/o2/o2119-005.git ||| ПОЯСНЕНИЕ: https://prnt.sc/rjvmcs - потом в созданной тобой папке появится папка с репозиторием.
